# BIP-39 English word list notice

`bip39-english.txt` is the 2,048-word English list published with Bitcoin Improvement Proposal 39. BIP-39 is licensed under the MIT License. The list is vendored unchanged so every implementation maps the same 11-bit values to the same four-word security phrase.

Upstream: <https://github.com/bitcoin/bips/blob/master/bip-0039/english.txt>
